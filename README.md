# Cinema Ticket App

Aceasta este o aplicatie pentru rezervare/cumparare bilete la film + administrare.

## Dependinte folosite

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
